102 Etiquetas HTML — Referencia rápida

Lista con número, etiqueta, descripción corta, ejemplo de uso y notas adicionales. Puedes copiar/pegar los ejemplos en tu index.html para probarlos.

# Etiqueta Descripción Ejemplo de uso Elemento adicional
1<a>Define un hipervínculo.<a href="https://www.example.com">Visita Ejemplo</a>Atributo target="_blank" abre en nueva pestaña.
2<abbr>Abreviatura con título explicativo.<abbr title="World Health Organization">WHO</abbr>Útil para accesibilidad y tooltip.
3<address>Información de contacto del autor o entidad.<address>Universidad Autónoma de Santo Domingo</address>Suele colocarse en el pie (<footer>).
4<area>Define un área sensible en un mapa de imagen (<map>).<area shape="rect" coords="34,44,270,350" href="mapa.html">Solo dentro de <map> y usado con <img usemap>.
5<article>Contenido independiente/autocontenido (post, noticia).<article><h2>Noticia</h2><p>Texto...</p></article>Ideal para entradas de blog o noticias.
6<aside>Contenido complementario (barra lateral).<aside>Enlaces relacionados</aside>Contenido no esencial para la lectura principal.
7<audio>Reproductor de audio en la página.<audio controls src="audio.mp3"></audio>Soporta múltiples <source> para formatos.
8<b>Texto en negrita sin significado semántico fuerte.<b>Texto importante</b>Para énfasis visual, preferir <strong> si es semántico.
9<base>Define URL base para enlaces relativos.<base href="https://midominio.com/">Va en <head> y solo un <base> por documento.
10<bdo>Sobrescribe la dirección del texto (dir).<bdo dir="rtl">Texto</bdo>Útil para idiomas con direcciones opuestas.
11<blockquote>Cita larga de otra fuente.<blockquote cite="fuente">Cita...</blockquote>Normalmente estilizado con margen.
12<body>Contenido visible del documento HTML.<body>Contenido de la página</body>Contiene todo lo que se renderiza.
13<br>Salto de línea (line break).Linea1<br>Linea2No usar para espaciado estructural.
14<button>Botón interactivo (forma o UI).<button type="submit">Enviar</button>type puede ser submit, reset o button.
15<canvas>Área de dibujo programático con JS.<canvas id="c" width="300" height="150"></canvas>Requiere JavaScript para dibujar contenido.
16<caption>Título de una tabla (<table>).<table><caption>Horario</caption>...</table>Debe ser primer hijo de <table>.
17<cite>Referencia al título de una obra.<cite>El Quijote</cite>No usar para la cita textual; para obras.
18<code>Fragmento de código en línea.<code>console.log('Hi')</code>Combina bien con <pre> para bloques.
19<col>Define propiedades para columnas de tabla.<col span="1" style="background:#eee">Se usa dentro de <colgroup>.
20<colgroup>Agrupa columnas de tabla (<col>).<colgroup><col span="2"></colgroup>Permite aplicar estilos por columna.
21<data>Asocia un valor legible por máquina a contenido.<data value="42">42</data>Útil para microdatos y parsers.
22<datalist>Lista de opciones para un <input list>.<datalist id="op"><option value="A"></datalist>No visible por sí solo; conectado a inputs.
23<dd>Descripción en lista de definiciones (<dl>).<dl><dt>HTML</dt><dd>Lenguaje de marcado</dd></dl>Complementa al <dt>.
24<del>Texto eliminado (tachado) con significado histórico.<del>Versión vieja</del>Puede llevar atributo datetime.
25<details>Sección plegable (ver/ocultar).<details><summary>Ver</summary><p>Contenido</p></details>Muy útil para FAQs.
26<dfn>Define la primera aparición de un término.<dfn>HTML</dfn> es...Semántico para glosarios.
27<dialog>Elemento de diálogo/modal (HTML5).<dialog open>Aviso</dialog>Controlable con JS para modalidades.
28<div>Contenedor de bloque genérico para agrupar elementos.<div class="card">Contenido</div>Muy usado para layout y estilos.
29<dl>Lista de definiciones (dt/dd).<dl><dt>Term</dt><dd>Definición</dd></dl>Útil para glosarios y FAQ detalladas.
30<dt>Término dentro de <dl>.<dt>HTML</dt>Va acompañado de <dd>.
31<em>Marca énfasis (normalmente cursiva).<em>Importante</em>Tiene significado semántico (énfasis).
32<fieldset>Agrupa controles en formularios.<fieldset><legend>Datos</legend></fieldset>Mejora accesibilidad y agrupación.
33<figcaption>Texto (pie) que describe una figura.<figure><img src="img.jpg"><figcaption>Pie</figcaption></figure>Se usa dentro de <figure>.
34<figure>Contenedor para imágenes/diagramas y su pie.<figure>...</figure>Permite asociar <img> con <figcaption>.
35<footer>Pie de sección o de la página.<footer>© 2025</footer>Suele incluir contacto o copyright.
36<form>Contenedor para inputs y envío de datos.<form action="/send" method="post">...</form>method puede ser GET o POST.
37<h1>Encabezado de nivel 1 — título principal.<h1>Mi título</h1>Importante para SEO; usar una por página.
38<h2>Encabezado nivel 2 — subsección.<h2>Sección</h2>Jerarquía: h1 → h2 → h3 ...
39<h3>Encabezado nivel 3.<h3>Subsección</h3>Usado para subtítulos internos.
40<h4>Encabezado nivel 4.<h4>Detalle</h4>Continuación de la jerarquía.
41<h5>Encabezado nivel 5.<h5>Nota</h5>Rara vez usado solo; mantiene estructura.
42<h6>Encabezado nivel 6 (más bajo).<h6>Pie de sección</h6>Último nivel de encabezados.
43<head>Contiene metadata y recursos (title, meta, link).<head><title>Mi sitio</title></head>No renderiza contenido visible.
44<header>Cabecera de sitio o sección (logo, nav).<header><h1>Logo</h1></header>Puede repetirse en secciones.
45<hr>Separador temático horizontal.<hr>Elemento vacío, usado para separación.
46<html>Elemento raíz del documento HTML.<!DOCTYPE html><html lang="es">...</html>Contiene <head> y <body>.
47<i>Texto en cursiva con uso tipográfico.<i>palabra</i>Preferible para términos en otro idioma.
48<iframe>Marco embebido que muestra otra página.<iframe src="https://example.com"></iframe>Atención a seguridad (sandbox, same-origin).
49<img>Inserta una imagen en la página.<img src="img/foto.jpg" alt="Foto">Siempre incluir alt para accesibilidad.
50<input>Campo de entrada para formularios.<input type="text" name="nombre">Soporta muchos tipos: email, number, checkbox...
51<ins>Texto insertado (nuevo).<ins>Versión nueva</ins>Complementa a <del> para cambios.
52<kbd>Texto que representa pulsaciones de teclado.<kbd>Ctrl</kbd>+<kbd>C</kbd>Útil en documentación o atajos.
53<label>Etiqueta asociada a un input (accesibilidad).<label for="n">Nombre</label><input id="n">Click en label focaliza el input.
54<legend>Título de un <fieldset> en formularios.<legend>Datos personales</legend>Mejora la comprensión del grupo.
55<li>Elemento de lista (item).<ul><li>Elemento</li></ul>Dentro de <ul> o <ol>.
56<link>Enlace a recursos externos (ej. CSS).<link rel="stylesheet" href="css/estilos.css">Va en <head>.
57<main>Contenedor del contenido principal del documento.<main>Contenido principal</main>Útil para accesibilidad y navegación por zona principal.
58<map>Mapa de imagen que contiene <area>.<map name="miMapa"><area ...></map>Se usa con <img usemap>.
59<mark>Resalta texto (marcado).<mark>Texto resaltado</mark>Visualmente marcado; útil en búsquedas.
60<meta>Metadatos del documento (charset, description).<meta name="description" content="Resumen">Importante para SEO y viewport.
61<meter>Indicador de un valor en un rango conocido.<meter value="0.6">60%</meter>No confundir con <progress>.
62<nav>Contenedor de enlaces de navegación.<nav><a href="/">Inicio</a></nav>Ayuda a motores y lectores a ubicar nav.
63<noscript>Mostrar contenido alternativo si JS está desactivado.<noscript>Activa JS para mejor experiencia</noscript>Se ejecuta solo si JS no está disponible.
64<ol>Lista ordenada (numerada).<ol><li>Primero</li></ol>Ideal para pasos o instrucciones.
65<optgroup>Agrupa opciones dentro de <select>.<optgroup label="Grupo"><option>A</option></optgroup>Mejora usabilidad de selects largos.
66<option>Opción dentro de un <select>.<option value="1">Uno</option>Puede llevar atributo selected.
67<output>Cuadro para mostrar resultados calculados.<output name="res">0</output>Puede actualizarse con JS.
68<p>Párrafo de texto (bloque).<p>Este es un párrafo.</p>Puede contener elementos en línea.
69<picture>Contenedor para imágenes responsivas (<source>).<picture><source srcset="img.webp" type="image/webp"><img src="img.jpg"></picture>Permite servir formatos/ resoluciones distintas.
70<pre>Texto preformateado que respeta espacios y saltos.<pre> texto\n salto</pre>Excelente para código sin formatear.
71<progress>Barra de progreso (tarea en curso).<progress value="70" max="100"></progress>Visual, no semántico para métricas complejas.
72<q>Cita corta en línea.<q>Una cita corta</q>Navegadores agregan comillas por defecto.
73<s>Texto tachado; indica obsolescencia visual.<s>Obsoleto</s>No implica eliminación histórica (use <del> para eso).
74<samp>Salida de programa o terminal.<samp>OK</samp>Semántica para mostrar resultados.
75<script>Incluye o linkea JavaScript.<script src="app.js"></script>Colocar al final de body mejora carga.
76<section>Sección temática del documento.<section><h2>Servicios</h2></section>Ayuda a estructurar contenido por temas.
77<select>Menú desplegable para elegir opción.<select><option>1</option></select>Puede ser single o multiple.
78<small>Texto de menor tamaño (notas).<small>Nota legal</small>Usar para disclaimers y créditos.
79<source>Fuente (archivo) dentro de audio/video/picture.<video><source src="video.mp4" type="video/mp4"></video>Permite declarar múltiples formatos.
80<span>Contenedor en línea genérico.<span class="res">texto</span>Ideal para estilos locales en texto.
81<strong>Texto con fuerte énfasis (importancia).<strong>Importante</strong>Semántica: más que solo estilo visual.
82<style>CSS embebido dentro del HTML.<style>body{margin:0}</style>Para CSS crítico; preferir archivos externos.
83<sub>Subíndice (texto bajo la línea).H<sub>2</sub>OÚtil en fórmulas químicas y notas.
84<summary>Título visible de <details>.<details><summary>Leer</summary></details>Click en summary despliega el details.
85<sup>Superíndice (texto arriba de la línea).x<sup>2</sup>Útil para exponentes y referencias.
86<table>Estructura de tabla para datos en filas/columnas.<table><tr><td>1</td></tr></table>Combinar con thead/tbody/tfoot para accesibilidad.
87<tbody>Cuerpo principal de la tabla (filas).<table><tbody>...</tbody></table>Separación semántica frente a thead/tfoot.
88<td>Celda de datos en una fila (<tr>).<tr><td>Dato</td></tr>Usar scope/headers si es necesario para accesibilidad.
89<template>Plantilla inactiva que no renderiza hasta JS.<template><div>Plantilla</div></template>Clonar con JS cuando se necesite renderizar.
90<textarea>Área de texto multilínea (formulario).<textarea rows="4"></textarea>Usar para comentarios o textos largos.
91<tfoot>Pie de tabla (resumen de columnas).<table><tfoot>...</tfoot></table>Se renderiza después del tbody.
92<th>Celda de encabezado en tabla.<tr><th>Encabezado</th></tr>Suele ser semántica y con scope para accesibilidad.
93<thead>Cabecera de la tabla (encabezados).<table><thead>...</thead></table>Separación útil para tablas grandes.
94<time>Representa fecha y/o hora legible y machine-readable.<time datetime="2025-09-11">11 Sep 2025</time>El atributo datetime es importante para parsers.
95<title>Título del documento que aparece en la pestaña del navegador.<title>Mi sitio</title>Va dentro de <head> y es clave para SEO.
96<tr>Fila de tabla que contiene <td> o <th>.<tr><td>1</td></tr>Único contenedor de celdas por fila.
97<track>Pistas de texto (subtítulos) para medios (<video>).<track kind="subtitles" srclang="es" src="subs.vtt">Soporta captions y descriptions.
98<u>Texto subrayado (predominantemente tipográfico).<u>Subrayado</u>Menor carga semántica; usar con cuidado.
99<ul>Lista desordenada (viñetas).<ul><li>Item</li></ul>Ideal para listados sin orden.
100<var>Representa una variable o nombre de entidad.<var>x</var> = 10Se estiliza tipográficamente distinto.
101<video>Elemento para reproducir video en la página.<video controls src="video.mp4"></video>Soporta múltiples <source> y subtítulos (<track>).
102<wbr>Indica un posible punto de quiebre de palabra.PalabraRomperAyuda a mejorar el wrap en textos largos o URLs.

Nota: los ejemplos están simplificados para demostración. Muchas etiquetas aceptan atributos adicionales que amplían su comportamiento (por ejemplo: aria-*, data-*, role, type, rel, target, etc.).
